# Advanced CSS Styling Project

## Overview

This project is the **CSS-focused continuation** of the HTML Advanced Project (0x02).  
While the previous project focused on structuring content with HTML, this project emphasizes **design, styling, and responsiveness** using CSS.

The goal is to transform an HTML page into a polished, visually appealing web page following the Figma design provided.

---

## Objectives

1. **Replicate Figma Design**  
   - Match typography, colors, spacing, and layout.  
   - Round float values where necessary.  

2. **Use Custom Fonts**  
   - `Source Sans Pro` for body text.  
   - `Spin Cycle OT` for headings or special elements.  
   - Include fonts via `@font-face` or Google Fonts links if applicable.

3. **Advanced CSS Techniques**  
   - Layout with Flexbox or CSS Grid.  
   - Hover effects, transitions, and interactive elements.  
   - Responsive design for multiple screen sizes.  

4. **Maintain HTML Structure**  
   - Copy `index.html` from the HTML Advanced Project (0x02).  
   - Use semantic HTML elements.  

5. **Accessibility & Best Practices**  
   - Ensure color contrast and readability.  
   - Include alt text for images.  
   - Ensure keyboard navigation works.
